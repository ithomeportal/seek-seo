# SEEK Equipment

SEO-optimized website for SEEK Equipment — trailer rental & leasing in Texas.

## Stack

- Next.js 16 (App Router)
- Tailwind CSS
- TypeScript
- Vercel deployment

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
